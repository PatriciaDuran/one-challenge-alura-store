# 🏬 Challenge Alura Store

## 📌 Descrição do Projeto

Este projeto tem como objetivo analisar o desempenho das quatro lojas da rede **Alura Store** para auxiliar na tomada de decisão sobre qual loja deveria ser vendida.

A análise foi realizada a partir de diferentes indicadores de desempenho, como faturamento total, vendas por categoria, produtos mais e menos vendidos, avaliação média dos clientes e custo médio de frete.

Com base nesses dados, foram geradas visualizações e análises que permitiram comparar o desempenho das lojas e identificar qual delas apresenta menor impacto financeiro para a rede.

---

## 🎯 Objetivo da Análise

Ajudar o Sr. João a tomar uma decisão estratégica sobre qual loja vender, considerando:

* Faturamento total das lojas
* Faturamento por categoria de produtos
* Produtos mais vendidos
* Produtos com menor faturamento
* Avaliação média dos clientes
* Custo médio de frete
* Comparação de desempenho entre as lojas

## 📈 Análises Realizadas

No projeto foram realizadas diversas análises para entender o desempenho das lojas da Alura Store. Entre as principais destacam-se:

* Faturamento total por loja
* Faturamento das catergorias mais lucrativas
* Produtos com maior e menor faturamento
* Avaliação média dos clientes por loja
* Relação entre preço dos produtos e avaliações
* Análise do frete médio por loja

Além dessas, outras análises complementares foram realizadas para apoiar as conclusões. Para facilitar a interpretação, todas as análises foram acompanhadas de visualizações gráficas que evidenciam padrões e diferenças entre as lojas.

---

## 📊 Principais Resultados

Após a análise dos dados, foi possível observar que:

* A **Loja 1** apresenta o maior faturamento.
* A **Loja 4** possui o menor faturamento entre as lojas.
* As categorias **Eletrônicos** e **Eletrodomésticos** concentram a maior parte da receita.
* A **Loja 4** não apresenta destaque relevante nos produtos mais lucrativos.
* Apesar de possuir frete médio menor, a Loja 4 continua apresentando desempenho inferior.

Com base nesses indicadores, a **Loja 4 foi identificada como a melhor candidata para venda**, pois sua retirada teria menor impacto na receita total da rede.
